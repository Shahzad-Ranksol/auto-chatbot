## MESSAGE TO REVIEWER

```
This is a self-hosted multi-tenant chatbot SaaS platform built with Node.js and Express.
It requires NO database server — all data is stored in a simple JSON file (data/db.json)
that is auto-created on first run. No MySQL, no PostgreSQL, no configuration needed.

Live demo: https://nimbleleads.co/chatbot-pro/install

Key points for review:
- Zero database setup — buyers just upload files, run npm install, and visit /install
- The /install route creates the first admin user and writes a lock file to prevent re-use
- The embedded widget uses a bundling-proof inline loader that works on WordPress and all CMS platforms
- No third-party API key is required — the built-in echo bot handles responses without one
- Works on cPanel shared hosting, VPS, and any Node.js host

Full documentation is included: README.md, INSTALLATION.md, USAGE.md, API.md, FAQ.md, CHANGELOG.md

Thank you for reviewing!
```
